# Microsoft 365 PowerShell Automation Lab

## Objective
This repository contains PowerShell scripts and administrative automation
used in a Microsoft 365 Enterprise lab environment.

## Environment
- Microsoft 365 Developer Tenant
- Microsoft Entra ID
- Exchange Online
- SharePoint Online
- Microsoft Teams

## Automation Areas

✔ User provisioning automation  
✔ License assignment  
✔ Role-based access control auditing  
✔ Conditional Access validation  
✔ Exchange mail flow testing  
✔ SharePoint governance checks  

## Sample Scripts (Planned)

- Connect to Microsoft Graph
- List all users with assigned licenses
- Export admin role assignments
- Check MFA status for privileged accounts

## Purpose

This lab simulates real-world enterprise administration tasks
for security hardening, identity protection, and compliance testing.
